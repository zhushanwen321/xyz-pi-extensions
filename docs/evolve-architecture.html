🎯 Problem Registry (src/problems.ts) 6 个 ProblemDefinition — 连接 Detector ↔ Analyzer Rule
compact-frequency
Compact 频率
context session_compact
≥3 次/session → high
rules: compact-high-frequency, compact-early-trigger
context-utilization
上下文利用率
context turn_end
>70% → medium, >90% → high
rule: context-high-utilization
subagent-efficiency
Subagent 效率
subagent tool_result
失败率 >20% → medium, >40% → high
rules: subagent-failure-rate, subagent-high-retry
tool-param-validation
工具参数校验失败
tool tool_result isError
>10% → medium, >25% → high
rules: param-error-rate, edit-match-failure, low-self-correction
workflow-phase-duration
工作流阶段耗时
workflow coding-workflow-gate
阶段占比 >50% → medium
rules: workflow-slow-phase, workflow-gate-retry
goal-task-quality
Goal 任务分解质量
workflow goal_manager
完成率 <70% → medium
rules: goal-low-completion, goal-stall-frequent, todo-high-abandon
