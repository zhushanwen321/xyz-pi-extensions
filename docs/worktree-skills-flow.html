📎 Custom Hooks 扩展点详情 (5 个 Hook + 环境变量)
setup-worktree.sh
git-cwt 创建后调用
xyz-agent: Electron 二进制缓存 (~242MB)、vendor 子模块复制、依赖安装
llm-simple-router: 依赖安装 + git hooks 配置
dag-executor: Python 虚拟环境 + 依赖安装
pre-merge.sh
merge 阶段 1 调用
llm-simple-router: 同步根 package.json 版本从 router/package.json
dag-executor / xyz-agent: 无此 hook
read-version.sh
merge 阶段 4 调用
llm-simple-router: 从 router/package.json 读取版本（根 pkg 无 version 字段）
其他项目: 无此 hook，默认从根 package.json 读取
post-bump.sh
merge 阶段 4 bump 后调用
llm-simple-router: 将新版本同步到所有子 package.json
xyz-agent: 同上，但跳过 resources/pi/ 下的第三方 package.json
generate-release-notes.sh
merge 阶段 5 调用
xyz-agent: 过滤 commit 清单，只保留 feat/fix/perf/breaking，排除 CI/build/chore
llm-simple-router: 无此 hook，使用默认规则
