① 卡片网格 — 架构概览

Gateway Layer

API Gateway

统一入口，路由、鉴权、限流。所有外部请求经此进入。

  • resolveRoute() 路由分发
  • authMiddleware JWT 校验
Port: 1 实现
Domain Layer

Order Aggregate

订单聚合根，包含状态机和不不变式校验。

  • 状态: created → paid → shipped
  • 不变式: 金额 ≥ 0
Aggregate Root
Infra Layer

PostgreSQL Adapter

持久化适配器，实现 OrderRepository Port。

  • save(order) 落库
  • findById(id) 查询
