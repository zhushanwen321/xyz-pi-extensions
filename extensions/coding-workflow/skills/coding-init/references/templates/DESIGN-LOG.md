## 状态语义

- `in-progress` — 设计/实施中，topic 目录可读写
- `archived` — coding-closeout 已收尾，topic 目录只读，沉淀已进长期文档
- `abandoned` — 放弃，标理由（沉淀仍可能有价值，归档前提取）
